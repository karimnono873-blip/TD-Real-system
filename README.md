# Correction TD Systèmes Temps Réel - Série 03

Ce dépôt contient la résolution détaillée des exercices de la Série 03 pour le module de **Systèmes Temps Réel**. La correction est présentée sous la forme d'une page web (HTML/CSS) dotée d'une interface moderne et épurée, inspirée du design **Google Pixel** (typographie Google Sans, nuances de bleu et cyan).

## 📋 Contenu du Projet

L'interface permet de consulter de manière claire et structurée :
* Les réponses expliquées pour chaque exercice.
* Les calculs des facteurs d'utilisation du processeur (U).
* L'analyse et les traces d'exécution (diagrammes de Gantt) pour les algorithmes d'ordonnancement suivants :
  * **RM** (Rate Monotonic) & test de Liu et Layland
  * **EDF** (Earliest Deadline First)
  * **LLF** (Least Laxity First)
  * **RMA / DMA** (Deadline Monotonic Assignment)
  * **Algorithmes classiques** : FCFS, Round Robin (RR), et SJF préemptif (SRTF)

## 🚀 Installation et Utilisation

1. **Cloner ou télécharger** ce dépôt sur votre machine locale.
2. **Ouvrir** le fichier `.html` directement dans votre navigateur web préféré (Chrome, Firefox, Edge, etc.). Aucune installation supplémentaire n'est requise.

*Astuce : Vous pouvez facilement héberger ce projet en ligne gratuitement en activant [GitHub Pages](https://pages.github.com/) dans les paramètres de votre dépôt.*
